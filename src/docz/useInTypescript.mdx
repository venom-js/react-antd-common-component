## 在Typescript中使用

如果你是 [venom](http://venom.org.cn) 用户，通过 `create-venom` 即可创建基于typescript的初始化工程，开箱即用。

## 安装create-venom使用

使用yarn安装
```shell
$ yarn global add create-venom
```

使用npm安装
```shell
$ npm i create-venom -g
```

查看当前模板
```
$ create-venom list
```

生成项目
```shell
$ create-venom init
```

创建项目后，cd进入项目，通过yarn或npm的方式安装相关依赖。

依赖安装成功后，通过yarn start或npm start启动项目。

启动成功后浏览器会自动访问 http://localhost:8000

typescript相关配置已经配置在生成的项目模板当中

## 在现有项目中使用

安装typescript相关依赖

```shell
$ yarn add typescript tslint-react tslint-config-airbnb tslint ts-jest-babel-7 --dev
``` 

如果你是react-hook用户，推荐安装 `tslint-react-hooks`

在根目录下创建tsconfig.json配置文件，以下配置以供参考：

```json
{
  "compilerOptions": {
    "target": "es6",
    "module": "es6",
    "moduleResolution": "node",
    "baseUrl": ".",
    "paths": {
      "src/*": ["./src/*"]
    },
    "sourceMap": true,
    "removeComments": true,
    "listEmittedFiles": true,
    "jsx": "react",
    "locale": "zh_cn",
    "allowSyntheticDefaultImports": true,
    "experimentalDecorators": true
  }
}
```
